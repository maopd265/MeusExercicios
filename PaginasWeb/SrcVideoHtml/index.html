Como adicionar vídeos do YouTube em páginas HTML

    1. Utiliza-se a tag iframe

< iframe width="560" height="315" src="https://www.youtube.com/embed/ii-KRRt13o4?si=FHPaeHocsScfXLpy" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen >

Para gerar o código embed com id do vídeo, você precisa:
    1.Acessar o vídeo no YouTube
    2.Ir em compartilhar
    3.Clicar em Incorporar
Após isso será gerado o código iframe na qual poderá ser utilizado na página html.
